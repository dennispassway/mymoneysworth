Door een van de markers op de kaart te selecteren verschijnt een informatiescherm met informatie over de valuta van de desbetreffende locatie. Het venster kan gesloten worden door op het kruisje te klikken. Ook is het mogelijk het venster te verplaatsen door de muis op het venster te plaatsen en deze te verslepen.

Wereldwijde Valuta's is een project van Dennis Passway. Het project draait op de Google Maps API en op de API van openexchangerates.org.

De pins zijn mogelijk niet zichtbaar in Firefox. Dit is een bekende fout in Firefox waar tot nu toe niets aan gedaan kan worden. Voor informatie: Firefox support forum.
